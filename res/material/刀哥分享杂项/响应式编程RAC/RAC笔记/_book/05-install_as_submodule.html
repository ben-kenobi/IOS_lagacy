手动安装 - Submodule

  • 新建 Xcode 项目
    • 创建项目时，一定要勾选 Git
  • 进入终端，输入以下命令
# 切换至项目目录
$ cd 项目目录

# 以子模块(submodule)的方式添加 ReactiveCocoa
$ git submodule add https://github.com/ReactiveCocoa/ReactiveCocoa.git submodule/ReactiveCocoa

# 递归更新 ReactiveCocoa 的依赖库
$ git submodule update --init --recursive
